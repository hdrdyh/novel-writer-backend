根据当前小说类型（玄幻长篇），统筹Agent建议以下Agent组合以获得最佳协作效果：

大纲阶段
统筹 → 编排流程
世界架构师 → 构建世界观
剧情设计师 → 设计情节线
统筹 → 整合报告
粗纲阶段
统筹 → 编排流程
世界架构师 → 补充设定

推荐理由

玄幻长篇需要完整的世界观和复杂的人物关系，建议启用世界架构师和人物设计师。写作阶段加入对话设计师和场景描写师可显著提升阅读体验，节奏把控师和风格润色师确保长篇质量稳定。伏笔设计师可在后续启用以增强叙事深度。
